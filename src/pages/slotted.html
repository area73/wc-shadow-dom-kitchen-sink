Working with slotted components

When we work with slots , the sloted HTML will not be inside shadow Root and there fore styling will need to be apply from the outside

In this example we have a webcomponent call wc-nav-bar.

The WC has a menu define inside the web component and a slot to place arbitrary html from parent

The problem that we are facing is that using the exact same markup, any thing that is inside the slot will not be styled with custom component styles

No matter how deep is your slote inside the WC that it will be place outside the shadow root of the WC, so you may spect unwanted style issues
